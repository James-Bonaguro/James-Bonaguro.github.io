# Create Project Tracker Database

## What to do
Set up a project tracker database with standard properties and a default view.

## Details
- Use the `project-tracker` template from `templates/`
- Properties: Name (title), Status (select), Priority (select), Due Date (date), Owner (person), Tags (multi-select)
- Status options: Not Started, In Progress, Blocked, Done
- Priority options: Low, Medium, High, Urgent
- Default view: table sorted by Priority descending, then Due Date ascending
