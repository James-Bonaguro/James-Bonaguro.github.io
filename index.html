Multi-AI Workflow

The actual system I use. Built this portfolio with it.

+
The idea

Different AI models are good at different things. So use them for what they're good at instead of forcing one model to do everything.

The process

GPT: Brainstorming, ideation, talking through messy ideas via voice

Claude (Opus 4.5): Actual building. Code, structure, the real work

Perplexity: Research, fact-checking, finding examples

Gemini: Deep research when I need to go down a rabbit hole

The trick

I have a template where I paste in the task and it generates prompts optimized for each model. Same task, different words. Currently improving this into something I might actually sell. Or not. We'll see.
